## Transformasi Layanan Cabang

Software kami dirancang khusus untuk memenuhi standar **Service Excellence** perbankan modern. Membantu manajemen cabang mencapai target **SLA (Service Level Agreement)** dan memastikan kepuasan nasabah tetap terjaga di jam sibuk.

### Fitur Utama

1.  **Integrasi Mobile Booking:** Nasabah dapat mengambil nomor antrian via aplikasi Mobile Banking sebelum datang ke lokasi (*Check-in* mandiri via QR Code di Kiosk).
2.  **Smart Flow Management:** Pemisahan otomatis jalur antrian untuk **Teller** (Transaksi Tunai), **Customer Service** (Non-Tunai), dan **Nasabah Prioritas/VIP**.
3.  **Audit KPI & SLA:** Data durasi pelayanan setiap staf terekam akurat. Memudahkan Kepala Cabang memantau produktivitas dan kepatuhan terhadap standar waktu layanan.
4.  **Head Office Dashboard:** Divisi Operasional Pusat dapat memantau volume antrian dan status layanan seluruh kantor cabang di Indonesia dalam satu layar.
5.  **Multi-Platform:** Berbasis Web (dapat diakses via PC Windows, Linux, atau Tablet Android oleh petugas Greeter).
